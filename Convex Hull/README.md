Brute Force:

    for element1 - O(N)
  
      for element2 - O(N)
    
        calculate which side - O(N)
      
  Total: O(N^3)
  
Graham Scan:

    find bottom left - O(N)
  
    sort in polar coordinates order - O(NlogN)
  
    stack process - O(N)
  
  Total: O(NlogN)
  
Divide & Conquer:
   
  Total: O(NlogN)
